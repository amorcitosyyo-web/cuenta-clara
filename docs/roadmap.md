# Cuenta Clara Roadmap

## Version actual

Esta version corre en el navegador y guarda datos localmente. Sirve para validar el flujo con ustedes antes de conectar usuarios reales.

## Version 1 compartida

- Login para cada persona.
- Una cuenta compartida por pareja.
- Sincronizacion en tiempo real con Supabase.
- Ingresos, gastos, presupuestos e historial.
- Roles simples: dueno y miembro.

## Version 2 facturas

- Subir fotos a Storage.
- OCR real para extraer fecha, comercio, total y productos.
- Confirmacion antes de guardar.
- Reglas aprendidas por comercio y producto.

## Version 3 analisis

- Reporte mensual.
- Comparacion contra meses anteriores.
- Alertas cuando una categoria se acerca al limite.
- Exportar CSV o PDF.

## Decisiones tomadas

- Moneda inicial: colones costarricenses.
- No se divide por persona porque usan una sola cuenta.
- Todo gira alrededor de ingresos, gastos y presupuesto compartido.
- Primero web responsive/PWA para que funcione en iPhone, Mac y PC.
